# minimal-alarm

シンプルなブラウザアラームアプリです。

## 使い方

1. `index.html` をブラウザで開く
2. 時刻を入力して **Set Alarm** を押す
3. 指定した時刻になるとビープ音と画面表示でお知らせ
4. **Dismiss** で停止

## 機能

- 現在時刻のリアルタイム表示
- アラーム時刻の設定・解除
- Web Audio API によるビープ音
- セット中はキャンセルボタンに切り替わる

## 動作環境

モダンブラウザ（Chrome / Firefox / Safari / Edge）であれば動作します。サーバー不要、`index.html` 単体で動きます。
